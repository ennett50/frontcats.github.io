Emmet — это инструмент для ускорения работы с HTML и CSS. Здесь будет подробно описаны все методы для написание быстрого html. Стоит напомнить, что все команды "разворачиваются" по клавише Tab

Базовый синтаксис

Дочерний элемент: >

nav>ul>li 

<nav>
  	<ul>
		<li></li>
	</ul>
</nav>

Следующий (дочерний) элемент: +

p+bq+span 

<p></p>
<blockquote></blockquote>
<span></span>

Подняться на один уровень в дереве HTML выше: ^

div>p>span^bq
Стоить отметить, что допустимы и множественные варианты. Например, p>a>span^^p 

<div>
    <p><span></span></p>
    <blockquote></blockquote>
</div>

Группировака по тэгам: ()

(div>p>span^bq)+ul>li 

<div>
    <p><span></span></p>
    <blockquote></blockquote>
</div>
<ul>
    <li></li>
</ul>

Умножение элементов: *

ul>li*5 

<ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
</ul>

id и class

#header+div.footer 

<span id="header"></span>
<div class="footer"></div>

Атрибуты тэгов: [ ]

a[title="Hello"] img[alt] 

<a href="" title="Hello"></a>
<img src="" alt="">

Заменяющий тект-рыба: Lorem

p>Lorem
Также допустимы значения по количеству слов. Например, Lorem5 

<p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Aliquam ducimus atque ut quia ratione, eveniet ipsum inventore provident! Enim quasi quo quia nam dolores est velit nihil delectus, sunt quisquam.</p>

Нумерация: ()

ul>li.item$*5
Вы также можете использовать множественный оператор$$$ 

<ul>
    <li class="item1"></li>
    <li class="item2"></li>
    <li class="item3"></li>
    <li class="item4"></li>
    <li class="item5"></li>
</ul>

Изменение нумерации и направления:
@-, @3

ul>li.item$@-*5
ul>li.item$@3*5 

 <ul>
    <li class="item5"></li>
    <li class="item4"></li>
    <li class="item3"></li>
    <li class="item2"></li>
    <li class="item1"></li>
</ul>
<ul>
    <li class="item3"></li>
    <li class="item4"></li>
    <li class="item5"></li>
    <li class="item6"></li>
    <li class="item7"></li>
</ul>

Текст: {}

a{click}+b{here}
a>{click}+b{here} 

<a href="">click</a><b>here</b>
<a href="">click<b>here</b></a>



Фильтры

Экранирует все html тэги: |e

#content>p.title|e 

<span id="content">
    <p class="title"></p>
</span>

Комментриует тэги: |с

div#page>p.title+p|c 

<div id="page">
    <p class="title"></p>
    <!-- /.title -->
    <p></p>
</div>
<!-- /#page -->

Все тэги пишет в одну линию: |s

p.title+p>span|t 

<p class="title"></p><p><span></span></p>

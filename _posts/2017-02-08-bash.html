Программное обеспечение

Здесь мы охватим базовые команды, без которых не обходится любой процесс разработки.

В начале мы рассмотрим, какое программное обеспечение нам нужно для работы. По операционным системам можно выделить:

  • MacOS - здесь можно использовать стандартный терминал, но также рекомендуют использовать iTerm
  • Windows - рекомендуют использовать cygwin или же консоль git`а, которое есть при установке
  • Linux - стандартный терминал

Основные команды

Файлы и директории

Показать текущюю директорию:
pwd 


:~$ pwd
:~$ /home/username

Переход в нужную дирректорию:
cd 


:~$ cd /home
:~/www$

Подняться на уровень выше:
cd . 


:~/www$ cd ..
:~/$

Перейти в домашную папку:
cd ~ 


:~/www/project$ cd ~
:~/$

Отобразить содержимое текущей директории:
ls 


:~/www/project$ ls
examples.desktop  www        Загрузки     Общедоступные
pgadmin.log       Видео      Изображения  Рабочий стол
redis-2.8.9       Документы  Музыка       Шаблоны

Создать дерево директорий:
mkdir [имя_директории] 


:~/$ mkdir test
:~/$

Создать дерево директорий:
mkdir -p /[имя_директории]/[имя_директории]/ 


:~/$ mkdir -p /tmp/test1/test2
:~/$

Удалить директорию:
rmdir [имя_директории]

Можно добавлять флаг -f, что удалить файлы форсировнно без лишних вопросов, например, rmdir -f test 

:~/$ rmdir test
:~/$

Удалить директорию со всеми вложенными папками и файлами(рекурсивно):
rmdir -r [имя_директории] 


:~/$ rmdir -r test
:~/$

Создать файл:
touch [имя_файла 


:~/$ touch README.md
:~/$

Удалить файл:
rm [имя_файла]

Можно добавлять флаг -f, что удалить файлы форсировнно без лишних вопросов, например, rm -f test 

:~/$ rm README.md
:~/$

Вывести содержимое файла в консоль:
cat [имя_файла] 


:~/$ cat README.md
#Lorem ipsum

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Expedita, et.

Скопировать содержимое файла в другой файл:
cp [имя_файла_1] [имя_файла_2] 


:~/$ cp README.md README_1.md 
:~/$

Скопировать содержимое директории в другую директорию:
cp -r [имя_директории_1] [имя_директории_2] 


:~/$ cp test test-1 -r
:~/$

Переименовать или переместить:
mv [имя_1] [имя_2]

Если [имя_2] существующий каталог - переместить [имя_1] в каталог [имя_2], иначе переименовать


:~/$ mv test1 test2
:~/$

Общее

Очистить консоль:
clear 


:~/$ clear
:~/$

Помощь:
help

Также можно запрашивать отдельные команды какhelp cd
